
		linux-dev-framework简介
		
======================

    linux-dev-framework是一个Linux程序开发框架，基于C语言和gcc工具链。 
    linux-dev-framework将库代码、应用程序代码、编译脚本进行了适当的组织。
    用户可以以此为基础，快速开始自己的Linux应用程序开发。
    
    本软件是一款开源、免费软件。
    具体版权说明见COPYING.txt。

    本软件的编译方法见Build.txt。
